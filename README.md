## Installing dependencies

To install the dependencies for the project, execute the following command in your virtual environment::

```bash
pip install -r requirements.txt
```

The `requirements.txt` file is generated using `uv` by executing the following command::

```bash
uv pip compile requirements.in -o requirements.txt
```

To add or edit dependencies, edit the `requirements.in` file and run the command above.
The existing `requirements.txt` will be edited to specify package versions for the new set of dependencies.
